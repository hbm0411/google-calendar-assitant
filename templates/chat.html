스타벅스 근무 일정 관리

AI와 대화하며 스타벅스 근무 일정을 관리하세요

안녕하세요! 스타벅스 근무 일정 관리 비서입니다. 근무 일정을 구글 캘린더에 자동으로 관리해드립니다.

사용 예시:
• "내일 오전 7시에 근무 일정 추가해줘" → "오픈"으로 자동 설정
• "다음 주 월요일 오후 2시 근무" → "마감"으로 자동 설정
• "오늘 일정 보여줘"
• 이미지와 함께 근무 일정을 보내주시면 더 정확하게 처리해드립니다

이미지 업로드 방법:
• 📎 버튼 클릭하여 이미지 선택
• Ctrl+V (또는 Cmd+V)로 이미지 붙여넣기
• 이미지 파일을 입력창에 드래그 앤 드롭

사용 팁:
• 이미지만 있어도 전송 가능
• 줄바꿈: Cmd+Enter (Mac) / Ctrl+Enter (Windows/Linux) 또는 Shift+Enter
• 전송: Enter
